# Basics Of Fighting 
MB1 to Attack
MB1Hold for Strong Attack
MB2 to Parry
Space to Jump 
Q/Shift to Roll
Hold Q/Shift to enter Run State
E for Weapon Ability / Interact
F to Lock Enemy Cam
RTY/123 for Spells
ZX to Switch Items
C to use Item
M for menu (pause game, switch weapons and armour, set spells, equip items)
Some weapons can't parry, some don't let you roll.
